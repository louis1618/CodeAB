# Code AB: 자동 백업 스크립트
**Code AB**는 여러분의 소중한 프로젝트를 정해진 시간마다 자동으로 백업해주는 프로그램입니다. 프로젝트 개발 시 한 번만 설정해두면 파일이 유실될 위험을 줄여줍니다.

---

### 주요 기능

* **자동 백업**: 지정된 시간 간격마다 자동으로 프로젝트를 백업합니다.
* **수동 백업**: 필요할 때마다 언제든지 백업을 수동으로 실행할 수 있습니다.
* **간편한 설정**: 백업할 프로젝트 폴더와 저장할 위치를 GUI를 통해 쉽게 설정할 수 있습니다.
* **로그 기록**: 모든 백업 이력이 로그 파일에 기록되어 작업 내용을 한눈에 확인할 수 있습니다.
* **에디터 필요 없음**: 모든 설정을 GUI에서 완료할 수 있습니다.

---

### 설치 및 실행 방법

1.  **설치 파일 다운로드**:
    * [Releases](https://github.com/louis1618/CodeAB/releases) 페이지에서 `CodeAB_Setup_v1.0.exe` 파일을 다운로드하세요.
2.  **설치 진행**:
    * 다운로드한 파일을 실행하고, 안내에 따라 설치를 완료하세요.
3.  **프로그램 실행**:
    * 설치 후 바탕화면이나 시작 메뉴에 생성된 바로가기를 통해 프로그램을 실행하세요.
4.  **초기 설정**:
    * 프로그램을 처음 실행하면 `설정` 탭에서 **백업 저장 폴더**와 **백업할 프로젝트 목록**을 설정해 주세요.
